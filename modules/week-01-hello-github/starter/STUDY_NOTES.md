## Git Workflow Understanding

**What is the difference between `git add`, `git commit`, and `git push`?**
[Explain each command and when you would use it]

Answer: First, use git add to move your modified files to the staging area, then use git commit to record those staged changes to a local repository with a description, then use git commit to push it to a remote repository such as github.

**Why do we create branches?**
[Explain the purpose of creating a student branch]
