## Git Workflow Understanding

**What is the difference between `git add`, `git commit`, and `git push`?**
[Explain each command and when you would use it]

Answer: "git add" stages changes and tells git what files were modified, "git commit" records staged changes to your
local repository and creates a permanent snapshot, and "git push" sends your local commits to a remote repository.
